# Flappy Bird 3D 🐦🎮

Классическая игра Flappy Bird, переосмысленная в трёхмерном пространстве. Проект создан с использованием **Three.js** и сборщика **Vite**.

[Играть онлайн] - (https://gpugs-dev.github.io/flappy-bird-3d/)

## Управление

- Пробел или ЛКМ — заставить птицу взлететь.
- Главная цель — пролетать между трубами и не врезаться в них.

## Особенности

- Полностью 3D-окружение с камерой, следящей за игроком.
- 3D-модели птицы, труб, монет, облаков и других объектов.
- Подсчёт очков и коллизии.
